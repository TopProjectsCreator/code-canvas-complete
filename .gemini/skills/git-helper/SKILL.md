# Git Maestro (git-helper)

## Automated Maintenance
- **Quick-Fixes**: Run `scripts/quick-fixes.sh [unlock|sync|cleanup]` for instant automated repository repairs.
- **Proactive Diagnostics**: Follow the `integrity-check.md` routine whenever an operation fails or the repository feels sluggish.

## Advanced Workflows
- **Bulk Commit Modification**: Use `commit-rewrite-guide` for batch-renaming commits or changing author info.
- **Rebase & History**: Comprehensive guides for resolution (`rebase-guide`) and power-user history manipulation (`advanced-workflows.md`).

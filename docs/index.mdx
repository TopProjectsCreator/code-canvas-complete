## Prompt-first AI workflow reference

Code Canvas Complete combines a browser IDE with AI copilots so your instructions become edits, builds, and uploads instantly. Whether you are outlining a hardware prototype, remixing media, or shipping a deployment, these docs guide you on how to talk to the assistant, steer the right tools, and keep every step moving without opening a terminal.

## Visual pipeline builder

The Automation canvas is a drag-and-drop workflow builder where you chain action blocks into executable pipelines. Each block does one thing — calling an API, running a query, transforming data, sending a notification, or invoking an AI model. Connect them in sequence, set up a trigger, and generate production code with one click.

## Creating your first automation

1. Open **Automation** from the sidebar
2. Click **New Pipeline**
3. Drag a **trigger block** onto the canvas first — this defines when the automation runs
4. Add action blocks and connect them with arrows
5. Configure each block with your settings (API keys, URLs, messages, etc.)
6. Click **Generate Python** or **Generate Node.js** to export the runnable code
7. Optionally save to a workflow for scheduled execution

## Trigger blocks

Every automation starts with a trigger that defines when it fires:

| Trigger | Description |
|---|---|
| **Schedule (Cron)** | Run at a set time — "Every night at 8 PM", "Every hour", or a custom cron expression |
| **Webhook** | Listens for incoming HTTP POST at a generated endpoint |
| **RSS Feed Monitor** | Polls a feed URL at a configurable interval |
| **Email Monitor** | Watches an inbox with subject/sender/keyword filters |
| **File Watcher** | Monitors a directory or Git path for changes matching a glob |
| **Database Change** | Fires on INSERT, UPDATE, or DELETE in a database table |
| **FTP Monitor** | Watches a remote FTP directory for new files |
| **Manual / Chat** | Run from the chat with a phrase like "Run my weekly report" |

When the first block is not a trigger, the block palette highlights trigger blocks with a warning to remind you to start with one.

## Variable interpolation

Reference the output of any previous block using double-brace tokens:

```
Hello {{prev.name}}, your order {{prev.order_id}} has shipped.
```

The code generator converts these to:

**Python:**
```python
f"Hello {prev.get('name', '')}, your order {prev.get('order_id', '')} has shipped."
```

**Node.js:**
```javascript
`Hello ${prev?.name ?? ''}, your order ${prev?.order_id ?? ''} has shipped.`
```

## Code generation

Click **Generate Python** or **Generate Node.js** to produce a complete runnable script:

- API keys are pulled from environment variables — secrets are never hardcoded
- Schedule triggers become cron-compatible main loops
- Each block output is passed as `prev` to the next block
- Mixed text/variable fields are wrapped in proper interpolation syntax
- Error handling is included with try/catch wrappers

## Why personalization is worth doing first

If the workspace is hard to read, everything feels harder than it needs to be. A good theme and readable text size can reduce eye strain and make day-to-day work feel smoother.

This guide is written for everyday users — no coding expertise required.

## Step 1: Choose a base theme

1. Open **Settings**.
2. Go to **Appearance**.
3. Try Light, Dark, and High Contrast.
4. Keep the one that feels easiest to read for 20+ minutes.

### Quick recommendation table

| Situation | Best starting point |
|---|---|
| Bright room / daytime | Light theme |
| Evening or low light | Dark theme |
| Accessibility-first setup | High contrast |

## Step 2: Fix text readability

Use this simple checklist:

- Increase editor text until it’s readable at a normal sitting distance
- Add line spacing if lines feel cramped
- Turn on word wrap for smaller screens
- Increase panel/UI scale if sidebars feel tiny

<Tip>
If you wear glasses, set text size for comfort first and only then choose colors.
</Tip>

## Step 3: Tune your layout

Most users like this starter layout:

- File tree pinned on the left
- Editor in center
- Terminal docked at the bottom
- Activity or AI panel on the right only when needed

You can always save this as a default profile and switch later.

## Accessibility settings average users often miss

- Enable stronger contrast
- Prefer larger click targets in dense panels
- Avoid low-contrast accent colors
- Reduce visual clutter (hide minimap if distracting)

## Theme setup for specific workflows

### For writing docs
- Lighter background
- Larger font
- Wider line spacing

### For debugging
- Dark background
- Distinct syntax colors
- Side-by-side diff view

### For presentations and demos
- Large UI scale
- High-contrast accents
- Minimal side panels

## Troubleshooting visual discomfort

If your eyes still feel tired after setup:

1. Increase font size one more step
2. Lower brightness of monitor/device
3. Switch to higher contrast theme
4. Reduce session length and take short breaks

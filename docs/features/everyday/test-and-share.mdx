## Test with real people early

A short 10-minute test with one real user often teaches more than hours of guessing.

## 3-task test script (copy this)

1. Start the project without instructions
2. Complete the main action
3. Explain one confusing step

## Questions to ask every tester

- What felt easiest?
- What was confusing?
- Did anything look broken?
- Would you share this with someone else?

## Safe sharing checklist

- Remove private keys and secrets
- Remove personal/sample sensitive data
- Share preview links, not admin credentials

## After feedback: what to do first

1. Group similar issues
2. Fix top 3 blockers
3. Re-test quickly
4. Publish improved build

## Levels should teach, not punish

Players stay engaged when each level introduces one new challenge at a time.

## Easy level progression model

- **Level 1:** Teach controls safely
- **Level 2:** Add one obstacle
- **Level 3:** Increase timing pressure
- **Level 4:** Combine previous mechanics
- **Level 5:** Offer a satisfying final challenge

## Design checklist for each level

- Clear objective shown on screen
- Fair starting position
- Consistent rules from prior level
- Reward moment at completion

## Common beginner mistakes

- Jumping from very easy to very hard too quickly
- Adding too many new mechanics at once
- Hiding critical paths with confusing visuals

## Better pacing tips

- Use checkpoints after difficult sections
- Add optional bonus routes for advanced players
- Keep mandatory path simple and readable

## Visual cues that help players

- Strong contrast for hazards
- Repeated icon style for pickups
- Consistent color meaning (green=safe, red=danger)

## Prompt-first platform

Code Canvas Complete lets you prompt the AI to edit files, run commands, build hardware, and collaborate without leaving the browser. This set of pages highlights the real tools you interact with, explains how to ask for what you need, and keeps every capability within easy reach.

## Developer reference

Need to extend the platform? Visit the developer reference for architecture diagrams, Supabase function details, CLI commands, and workflow hooks so you can customize the experience while still keeping the AI workflows friendly.

## How to use this site

Each page in the **Product** tab explains a feature block in depth. Start with the IDE page to understand the core workspace, then explore AI, editors, execution, collaboration, and deployment in sequence. All the content is tied back to the source code under `src/`, `supabase/functions/`, and `tools/` so you can follow along with the implementation.

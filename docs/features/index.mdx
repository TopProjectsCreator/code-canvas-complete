## The all-in-one browser IDE

Code Canvas Complete is a professional-grade IDE that runs entirely in the browser. It combines multi-language code editing, AI-powered coding assistance, hardware programming, visual block coding, 3D model generation, audio production, office document editing, and real-time collaboration — all in a single workspace with no local installation required.

Whether you're shipping a web app, flashing an Arduino board, building a Scratch game, or generating a 3D model, Code Canvas Complete gives you the tools and the AI support to do it.

![While you wait for builds](while-you-wait.png)

## What you can build

- **Full-stack web apps** — React, Vue, Svelte, Node.js, Python, Go, Rust, and 30+ other runtimes
- **Hardware firmware** — Arduino (AVR/ESP), FTC robot code (Java/Kotlin), breadboard simulation
- **Visual programs** — Scratch block coding with live preview and export
- **3D models** — AI-powered generation with Meshy, Tripo, and Fal.ai integrations
- **Office documents** — Word (.docx), Excel (.xlsx), and PowerPoint (.pptx) editing with AI summaries
- **CAD designs** — View and explore 3D CAD models (.stl, .obj, etc.)
- **Media projects** — Video and audio editing with FFmpeg in the browser

## Platform sections

### Core workspace

Everything you need for professional code editing lives in the main IDE workspace — tabs, splits, file tree, terminal, Git panel, package manager, environment secrets, and a global search bar.

### AI assistant

The multi-model AI assistant understands the full context of your project. It can edit files, run commands, explain symbols, and generate code. Compare responses from OpenAI, Anthropic, and Gemini side-by-side, then pick the best one.

![Easy template changing with AI](easytemplatechanging.png)

### Specialized editors

Code Canvas includes dedicated editor suites for domains beyond text code:

- **Arduino IDE** with breadboard designer, simulator, and OTA/Bluetooth uploads
- **FTC Robotics** with Java/Kotlin compilation
- **Scratch** visual block programming with live canvas
- **3D model viewer** and AI generation
- **Video and audio editor** powered by FFmpeg
- **Word, Excel, PowerPoint** in the browser
- **CAD model viewer** for .stl, .obj, and .glb files

### Collaboration

Invite teammates, start WebRTC audio/video rooms, pair-program with live cursors, record sessions, and review AI spend per user — all from the collaboration panel.

### Workflows & automation

Build repeatable automation sequences that chain AI prompts, terminal commands, API calls, and file operations. Trigger them manually, on a schedule, or from chat.

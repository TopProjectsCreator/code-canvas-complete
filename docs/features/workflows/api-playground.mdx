## Compose requests

The API playground lives inside the workspace so you can craft headers, payloads, and query params without leaving the project. Save successful requests as templates and ask the assistant to reuse them inside workflows.

## Use AI to fill payloads

The assistant reads OpenAPI specs and uses schema-aware prompts so you don’t have to guess required fields. Ask "Call the deployments endpoint" and it will populate the body, headers, and auth automatically.

## Workflow integration

Drag saved templates into workflow steps or embed inline HTTP calls inside chat instructions. Each API step records the URL, method, headers, body, and response so you can replay the run or rerun just that call.

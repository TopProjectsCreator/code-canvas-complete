title: "Workflow history"
description: "View runs, logs, and rerun failed steps." 
---

## Run list

The history log lists every workflow run with the time, the user who triggered it, and the triggers that fired. Each entry links back to the workflow definition and shows whether it succeeded, failed, or is still running.

## Step logs

Drill into any run to see the logs produced by each step. You can copy the output, rerun a single step, or have the assistant summarize the failure if something timed out.

## Sharing history

Pin a run ID in chat or attach it to a support ticket so teammates can jump straight to the logs and see what happened without re-running anything.

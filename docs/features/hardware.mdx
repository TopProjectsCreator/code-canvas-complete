## Embedded development in the browser

Code Canvas includes a complete hardware development environment for Arduino and FTC robotics. Open the hardware panel, select your board, write your sketch, simulate it, and upload to the physical device — all without installing the Arduino IDE locally.

## The hardware workflow

```
Design circuit → Write sketch → Compile → Simulate → Upload to board
```

Each step has dedicated tooling inside Code Canvas:

| Step | Tool |
|---|---|
| Design circuit | Breadboard drag-and-drop designer |
| Write sketch | .ino editor with AI and syntax highlighting |
| Compile | Godbolt-powered server-side compilation |
| Simulate | Virtual breadboard with component simulation |
| Upload | OTA (ESP) or Bluetooth (AVR) upload bridge |

## Breadboard designer

The breadboard designer is a visual circuit design canvas:

- Drag components (LEDs, resistors, buttons, sensors, displays) from the palette
- Connect pins with click-to-draw wires (auto-colored by signal type)
- See component datasheets inline by hovering over any part
- Export the design as a wiring diagram image for documentation

## AI-assisted sketch writing

The AI assistant is deeply integrated with the Arduino editor:

![Arduino editor with AI assistance](ardurino-with-assistant.png)

- **"Add a buzzer that beeps on button press"** — inserts the complete code block
- **"Read temperature every second and print to serial"** — adds DHT library, setup, and loop logic
- **"Explain what this setup() function does"** — gives a plain-English breakdown
- **"Fix this compilation error"** — reads the error, diagnoses the cause, and proposes a fix

## Supported boards

### AVR (Bluetooth upload)
- Arduino Uno, Nano, Mega, Leonardo, Micro

### ESP (OTA upload)
- ESP32, ESP32-S2, ESP32-S3, ESP32-C3
- ESP8266, Wemos D1 Mini, NodeMCU

### FTC Robotics
- REV Control Hub and Expansion Hub (Java/Kotlin)

See [Supported boards](/features/ide/specialized-editors/arduino/supported-boards) for the full list.

## Toolchain health check

Before a compile or upload, Code Canvas checks whether the required toolchain tools are available:

- `avrdude` — AVR upload tool
- `esptool` / `espota.py` — ESP upload tools
- Board libraries and headers

If any tool is missing, the workspace displays a clear warning with a link to the required setup steps. The AI can guide you through installation interactively.

## FTC Robotics

Open the FTC panel for FIRST Tech Challenge development:

- Java and Kotlin code editor with FTC SDK 10.x support
- OpMode templates (Autonomous and TeleOp) with hardware map stubs
- Compilation and code verification before pushing to the robot
- Documentation links to FTC SDK API reference inline

## Visual builder

Drag blocks to model logic, state machines, or robotics behavior. Each block snaps into place, and you can preview the animation instantly.

## Export to code

When the block flow is ready, ask the assistant "Convert to JavaScript" or "Generate Arduino code" and the system creates the corresponding script in your project.

## Collaboration

Share the Scratch tab with teammates so everyone can edit the same blocks. Inline notes document what each block does for people reviewing the logic later.

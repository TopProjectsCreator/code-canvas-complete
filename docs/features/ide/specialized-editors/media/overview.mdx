## Video and audio studio

Trim clips, adjust audio, and apply filters. Ask the assistant to "shorten this clip to 30 seconds" or "add background music" and it edits in place.

## 3D editor & text-to-3D

Manipulate objects, change materials, and trigger text-to-3D generation from multiple providers. The workspace keeps lighting, textures, and render settings close at hand.

## Asset management

Each media editor saves its state with the project, so collaborators can open the same session and continue editing.

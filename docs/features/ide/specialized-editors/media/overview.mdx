## Creative tools in your IDE

The Media editor panel brings video, audio, image generation, and 3D model tools directly into the Code Canvas workspace. You don't need Premiere Pro, Audacity, or Blender — the tools live alongside your code.

## 3D model generation

Generate 3D models from text descriptions using multiple AI providers:

| Provider | Specialization |
|---|---|
| **Meshy** | Game-ready assets, textures, low-poly |
| **Tripo** | Photorealistic models from image or text |
| **Fal.ai** | Fast experimental 3D generation |
| **Neural4D** | Animated and rigged character models |

### How to generate a model

1. Open the **3D** panel from the sidebar
2. Type a description: `"A cartoon robot with round wheels and LED eyes"`
3. Choose a provider and quality setting
4. Wait for generation (typically 30–90 seconds)
5. Preview the model in the 3D viewer
6. Download as .glb or .obj for use in game engines or printing

## 3D model viewer

Import and inspect existing 3D files:

- **Supported formats** — .stl, .obj, .glb, .gltf, .fbx
- **Controls** — orbit (left drag), pan (right drag), zoom (scroll)
- **Views** — solid, wireframe, UV grid, and material override modes
- **Lighting** — ambient, directional, and HDRI environment maps
- **Annotations** — add labels and dimension markers
- **Fullscreen** — expand the viewer for detailed inspection

## Video editor

Edit video clips directly in the browser using FFmpeg:

- **Import** video files (.mp4, .webm, .mov, .avi)
- **Timeline** — arrange, trim, cut, and split clips
- **Audio track** — separate audio from video, adjust volume and fade
- **Filters** — brightness, contrast, saturation, and blur
- **Text overlay** — add captions and titles with font control
- **Export** — render to .mp4, .webm, or .gif

### AI video prompts

- **"Trim this clip to 30 seconds"** — auto-cuts to the most relevant portion
- **"Add background music"** — generates or selects ambient audio
- **"Create a title screen"** — adds an intro slide with your project name

## Audio editor

Edit and generate audio:

- **Waveform editor** — visual audio editing with cut, copy, and paste
- **Effects** — fade in/out, normalize, pitch shift, noise reduction
- **Recording** — record from your microphone directly into the editor
- **AI music generation** — generate background music via Lyria (Stable Audio)
- **Export** — save as .mp3, .wav, .ogg, or .flac

## Image generation

Generate images and assets for your projects:

| Provider | Type |
|---|---|
| **Stability AI (SDXL)** | Photorealistic, artistic images |
| **Ideogram** | Text-in-image, typography |
| **Replicate** | Open-source models (Flux, SDXL, etc.) |
| **Fal.ai** | Fast real-time image generation |

Use generated images as:

- UI component backgrounds or illustrations
- Sprite sheets for game development
- Slide thumbnails for presentations
- Product mockups for documentation

## Asset management

All media files created or imported in the editor are saved to your project's asset store:

- Files persist with the project across sessions
- Collaborators in a shared session see the same asset library
- AI can reference project assets by name when generating new content

## Microsoft Office in your browser

Code Canvas includes built-in editors for Word (.docx), Excel (.xlsx), and PowerPoint (.pptx) files. Create, view, and edit Office documents alongside your code without installing Microsoft Office or switching apps.

## Word document editor

![Word document editor](word.png)

The Word editor provides a familiar rich text experience:

- **Formatting** — headings (H1–H6), bold, italic, underline, strikethrough, and highlight
- **Lists** — ordered and unordered lists with nesting
- **Tables** — create, resize, and style data tables
- **Images** — embed images from your project assets or URL
- **Export** — save as .docx for use in Microsoft Word or Google Docs

### AI assistance in Word

- **"Write an executive summary"** — generates a structured summary of your project
- **"Rewrite this paragraph in formal tone"** — adjusts writing style
- **"Create a table of contents"** — scans headings and builds the TOC automatically
- **"Translate to Spanish"** — translates selected text to another language
- **"Summarize this document"** — produces a concise bullet-point summary

## Excel spreadsheet editor

![Excel spreadsheet editor](excel.png)

The Excel editor supports:

- **Formulas** — `SUM`, `AVERAGE`, `IF`, `VLOOKUP`, `INDEX/MATCH`, and more
- **Cell formatting** — number formats, currency, percentages, dates
- **Conditional formatting** — highlight cells based on rules
- **Multiple sheets** — add, rename, and switch between worksheets
- **Export** — save as .xlsx compatible with Microsoft Excel and Google Sheets

### AI assistance in Excel

- **"Add a formula to calculate the total in column D"** — inserts the correct formula
- **"Create a chart from this data"** — generates a chart recommendation with the right type
- **"Explain what this formula does"** — plain-language description of complex formulas
- **"Fill in missing values"** — intelligently completes patterns in your data

## PowerPoint presentation editor

![PowerPoint presentation editor](powerpoint.png)

The PowerPoint editor lets you:

- **Create slides** with text boxes, shapes, images, and icons
- **Apply layouts** — title, content, two-column, blank, and more
- **Add transitions** and animations
- **Themes** — change the color palette and font style globally
- **Speaker notes** — add notes per slide for presentation reference
- **Export** — save as .pptx for use in PowerPoint, Keynote, or Google Slides

### AI assistance in PowerPoint

- **"Create a 5-slide deck about our product"** — generates slides with titles and content
- **"Suggest a color theme"** — recommends a palette based on your branding
- **"Write speaker notes for slide 3"** — drafts talking points from the slide content
- **"Add an image placeholder to each slide"** — modifies layout structure

## Opening Office files

Office files can be:

1. **Uploaded** from your local machine via the file tree
2. **Created** using the new file dialog (choose .docx, .xlsx, or .pptx)
3. **Linked** from your project's assets directory

The editor auto-detects the file type and opens the appropriate editor. You can have a Word doc and a Python file open in split view simultaneously.

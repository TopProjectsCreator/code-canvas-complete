## Document creation

Start with templates or blank documents, then ask the assistant for outlines, summaries, or tables. The Word editor supports comments, the Excel sheet handles formulas, and PowerPoint can import visuals from the AI.

## Collaboration

Share docs with teammates, drop inline comments, and ask the AI for the next steps. Context pins keep every document section linked to related code or editors.

## Prompt guidance

Tell the assistant what tone or format you need (e.g., "Budget report for executives"), and it adjusts headings, bullet styles, and imagery to match.

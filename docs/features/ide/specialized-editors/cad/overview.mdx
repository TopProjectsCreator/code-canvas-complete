## CAD modeling

Sketch parametric shapes, tweak dimensions, and ask the AI for quick adjustments. Export previews, share with teammates, or send to CAM tooling.

## Database design

Use the database designer to create tables, define relationships, and generate SQL. The AI explains constraints in plain language and can run migrations via workflows.

## Prompts and reuse

Save templates for recurring models or schemas, and trigger them from workflows so collaborators can spin up new diagrams with a single command.

title: "Coding in .ino"
description: "Write and prompt for Arduino sketches." 
---

Use the .ino editor with syntax highlighting, automatic formatting, and inline AI suggestions. Ask the assistant to "Add PWM control to pin 5" and it inserts the function, comments, and setup wiring block.

### Sketch hints

- The assistant keeps track of `setup()` and `loop()` so it won’t rewrite the same initialization twice.
- Use the AI comparison view to review edits across models before accepting them.
- When you save, the `.ino` file is compiled automatically for the next upload attempt.

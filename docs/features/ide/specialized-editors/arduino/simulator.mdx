---
title: "Arduino simulator"
description: "How the virtual breadboard and component simulator work." 
---

Drag components onto the virtual breadboard to prototype sensors, LEDs, and modules. The simulator mirrors pin wiring and lets you test logic before compiling. Hover over parts to see properties and use the timeline to simulate step-by-step.

The assistant can summarize the simulation steps if you ask “What happens when I trigger the button?” so you always know what state your hardware is in before flashing code.

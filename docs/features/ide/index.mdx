## Everything in one place

The Code Canvas IDE is a full desktop-class development environment running in your browser. No downloads, no setup — open a project and start writing code immediately. It handles large repos, complex refactors, terminal workflows, Git operations, and package management as naturally as VS Code or JetBrains tools.

## Editor features

### Tabs, splits, and layouts

- **Tabbed editing** with dirty indicators (●) shows unsaved files at a glance
- **Split panes** — drag any tab to create horizontal or vertical splits
- **Custom layouts** — save and restore your split configuration per project
- **Minimap** gives a zoomed-out view of the file for fast navigation
- **Sticky scope header** anchors the current function or class name while you scroll through long files

### Multi-cursor and advanced editing

- **Multi-cursor** (Alt+Click or Ctrl+D for next match) for simultaneous edits
- **Column/block selection** for aligned data editing
- **Regex find-and-replace** with live match preview before committing
- **Bracket matching and auto-close** for all supported languages
- **Auto-import suggestions** for TypeScript, JavaScript, Python, and Java

### Global search

The command palette (Ctrl+P) lets you:

- Jump to any file by name
- Run editor commands
- Trigger AI prompts from the keyboard
- Switch active themes
- Open settings dialogs

## Large repository support

Code Canvas handles large codebases efficiently with lazy file loading and virtual scrolling.

![Large repos handled efficiently](largerepos.png)

The file tree supports:

- Recursive folder expand/collapse
- .gitignore-aware filtering
- File renaming, deletion, and drag-to-move
- New file/folder creation with template selection
- Right-click context menus for quick actions

## Terminal and shell

The built-in WebContainer terminal runs real Node.js natively in the browser:

- Run `npm`, `pnpm`, `yarn`, and `bun` package managers
- Execute Node.js scripts, shell scripts, and CLI tools
- Supports multiple terminal tabs
- Color-coded output with scrollback
- Keyboard shortcuts: Ctrl+` to toggle, Ctrl+Shift+C to copy

For Python, Java, Go, C++, and other languages, Code Canvas routes execution through Wandbox for server-side compilation.

## Git integration

![Git initialized and ready](git-initalized.png)

The Git panel is always accessible from the sidebar:

- View staged, unstaged, and untracked file diffs
- Stage individual files or hunks
- Commit with a message (AI can suggest commit messages from the diff)
- Push/pull from GitHub, GitLab, or Bitbucket remotes
- Switch branches, create new branches, and merge
- Resolve merge conflicts with a visual 3-way diff

## Package manager

The Packages panel shows installed npm packages with version info. You can:

- Search and install new packages
- Update or remove existing ones
- View dependency graphs
- See bundle size impact estimates

![Dependencies and packages panel](dependencies.png)

## Environment & secrets

All environment variables are stored encrypted per scope:

| Scope | Description |
|---|---|
| **Preview** | Used during development and previewing |
| **Shared** | Available in all environments |
| **Production** | Only active after deployment |

Secrets never appear in AI chat transcripts. When the assistant reads a secret to perform a task, it accesses the value securely without exposing it.

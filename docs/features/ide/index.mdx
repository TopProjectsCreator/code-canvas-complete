## Prompt-first code editing

The workspace is built around a promptable editor. Tell the AI which file or component you want to change, and it will highlight the suggested edits before you accept them. You can still navigate files manually with tabs and splits when you prefer a direct touch.

### Tabs, splits, and navigation

- **Tabbed editing** keeps each file accessible so you can switch contexts without losing focus.
- **Split views** let you pin documentation, terminals, and editors side-by-side. Drag tabs to create custom layouts and save them for later.
- **File tree and minimap** keep the structure of large workspaces visible, while sticky scopes anchor your viewport when scrolling.

### Global search and multi-tool access

- The universal search bar lets you jump to files, run commands, or trigger AI prompts from a single place.
- **Multi-cursor column editing** handles aligned data, and **regex replace** previews the impact before you commit.
- **Dirty indicators, context menus, and tab groups** behave like a desktop IDE but run entirely in the browser.

## Built-in terminals, git, and packages

The WebContainers-powered terminal runs `npm`, `pnpm`, `git`, and custom scripts without extra setup. The Git panel keeps you aware of branches, unstaged changes, and diffs, while provider integrations (GitHub, GitLab, Bitbucket) sync your remote repositories automatically.

### Environment manager & secrets

The environment dashboard stores secrets securely per scope (Preview, Shared, Production). When you ask the AI to read or write a secret, it accesses only the encrypted value, so keys never leak into the chat transcript.

<Note>
  Panel layout, terminal history, and package state persist in your browser session, so you can reload without losing context.
</Note>

## Extensions and how they work

The IDE includes an extensions system for creating custom tools, automations, and assistant helpers.

- **Widget extensions** render mini apps inside the IDE UI.
- **Command extensions** transform active file content.
- **Chat-tool extensions** add focused helpers for AI workflows.

Use the dedicated extensions guide for lifecycle diagrams, runtime behavior, and practical patterns.

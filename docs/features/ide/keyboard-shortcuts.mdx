## Why shortcuts matter

Code Canvas is designed for keyboard-first workflows. Learning a small core set will dramatically reduce context switching and speed up day-to-day development.

## Core shortcuts (Windows / Linux)

| Action | Shortcut |
|---|---|
| Command palette | `Ctrl+P` |
| Global search | `Ctrl+Shift+F` |
| Toggle terminal | `Ctrl+\`` |
| Save file | `Ctrl+S` |
| New file | `Ctrl+N` |
| Close tab | `Ctrl+W` |
| Next tab | `Ctrl+Tab` |
| Previous tab | `Ctrl+Shift+Tab` |

## Core shortcuts (macOS)

| Action | Shortcut |
|---|---|
| Command palette | `Cmd+P` |
| Global search | `Cmd+Shift+F` |
| Toggle terminal | `Cmd+\`` |
| Save file | `Cmd+S` |
| New file | `Cmd+N` |
| Close tab | `Cmd+W` |
| Next tab | `Ctrl+Tab` |
| Previous tab | `Ctrl+Shift+Tab` |

## Editing shortcuts

| Action | Windows / Linux | macOS |
|---|---|---|
| Multi-cursor add | `Alt+Click` | `Option+Click` |
| Select next occurrence | `Ctrl+D` | `Cmd+D` |
| Duplicate line | `Shift+Alt+Down` | `Shift+Option+Down` |
| Move line up/down | `Alt+Up/Down` | `Option+Up/Down` |
| Format document | `Shift+Alt+F` | `Shift+Option+F` |
| Rename symbol | `F2` | `F2` |

## AI and workflow shortcuts

| Action | Suggested shortcut |
|---|---|
| Open AI panel | `Ctrl/Cmd+Shift+A` |
| Apply AI patch preview | `Ctrl/Cmd+Enter` |
| Ask AI about selected code | `Ctrl/Cmd+K` |
| Run last workflow | `Ctrl/Cmd+Shift+R` |

<Note>
Teams often customize these bindings. If a shortcut doesn't match your setup, open Settings → Keybindings and search by action name.
</Note>

## Terminal shortcuts

| Action | Windows / Linux | macOS |
|---|---|---|
| Copy selection | `Ctrl+Shift+C` | `Cmd+Shift+C` |
| Paste | `Ctrl+Shift+V` | `Cmd+Shift+V` |
| New terminal tab | `Ctrl+Shift+\`` | `Cmd+Shift+\`` |
| Clear terminal | `Ctrl+L` | `Cmd+K` |
| Interrupt process | `Ctrl+C` | `Ctrl+C` |

## Git shortcuts

| Action | Windows / Linux | macOS |
|---|---|---|
